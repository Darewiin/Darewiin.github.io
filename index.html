AWK Lab — transform.awk

Standalone awk script from the CIS-245 Awk Lab. Processes colon-delimited data to filter by last name, contribution ranges, and computed averages using pattern matching, split(), and arithmetic.

transform.awk AWK 
BEGIN { FS = ":" }

# (a) Print first name of all Tooks followed by total contributions
/Took/ {
    split($1, a, " ")
    total = $3 + $4 + $5
    print a[1], "$"total
}

# (b) Print full names and contributions of anyone between $10 and $200 last contribution
$5 >= 10 && $5 <= 200 {
    print $1, "$"$5
}

# (c) Print full names and average contribution of those who contributed less than $300 on average
{
    avg = ($3 + $4 + $5) / 3
}
avg < 300 {
    print $1, "$"avg
}

AWK Lab — Selected Commands

Highlights from the full 20-question awk lab. These examples show pattern matching with regex, multi-word name handling via split(), area code extraction, and conditional filtering.

awk-lab-highlights.sh Bash + AWK 
# Print last names starting with T or D (handles middle names)
awk -F: '{n=split($1, a, " "); if (a[n] ~ /^[TD]/) print a[n]}' AwkLab.data

# Print names and area codes for those NOT in the 916 area code
awk -F: '$2 !~ /\(916\)/ {n=split($1, a, " "); split($2, b, ")"); print a[n], b[1] ")"}' AwkLab.data

# Print first names, last initial, and avg contribution (avg > $300)
awk -F: '{avg=($3+$4+$5)/3} avg > 300 {n=split($1, a, " "); print a[1], substr(a[n],1,1), "$"avg}' AwkLab.data

# Change a name and rebuild the line preserving colon delimiters
awk -F: '{if ($1 == "Samwise Gamgee") $1="Sean Astin"; print $0}' OFS=: AwkLab.data

# Print first names and total contributions (< $700 over 3 months)
awk -F: '{total=$3+$4+$5} total < 700 {n=split($1, a, " "); print a[1], a[n], "$"total}' AwkLab.data
